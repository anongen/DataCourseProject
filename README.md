# DataCourseProject
Course Project requirement

This repo includes the requirements for the “Getting and Cleaning Data course”.
There are two files:
1)	Run_analysis.R
The code used to take the initial data for the project and produce tidy_data.txt

2)	 CodeBook.md
A file that contains descriptions of the variables, the data, and the transformations that were performed to clean up the data and produce tidy_data.txt
